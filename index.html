Cloudflare Image Crop Adjustment

This tool allows you to experiment with Cloudflare's image cropping and gravity features by generating transformation URLs based on your input. You could also save the cropped image in a specific format (Do not use automatic format for this).

Virtual zoom means that the trim and crop are both adjusted to create a zoom effect.

Gravity can use auto and face detection. They usually give good results, but there is a custom option as well.
